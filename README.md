## 0.数学（143）

**资料**
（1）李永乐：全书（基础篇）+（综合提升篇），660题；（2）武忠祥：高数讲义（严选题没来得及做，听说题目出得很好，有点亏吧算是）；（3）李林：880（提升篇）+108；（4）真题：写了1995-2021；（5）模拟题：22李林6+4，22李永乐6+3，李艳芳三套卷（22和21），21合工大超越五套卷。

**3月中旬-4月初**：过了一遍全书（基础篇），并刷完了上面的习题。个人感觉，基础篇的题目难度远低于考研，对个人提升不大，我建议基础不是很差的话直接综合提升篇就好。

**4月中-7月中**：（有期末考试和自己专业课设计性实验等，这段时间进度较慢） （1）李永乐全书综合提升篇，综合提升篇每一道例题建议搞懂，学完一章后认真独立完成配套的练习，对于例题和习题中的错题、疑难点会进行二刷。（2）660题，高数部分选择题我觉得还是蛮好的，注重概念的理解，今年考研选择题也体现了这个趋势。660题是与全书综合篇穿插进行的，没有固定进度，看心情往下推进。

**7月中-9月底**：（1）这个阶段每天4-6小时，看高数辅导讲义，很难的地方(中值定理的证明题、定积分）从21年的武忠祥老师17堂课挑选对应的部分看一下视频。（2）880题提升篇，质量很高，高数难度中上，线代偏难，做的时候被卡住很正常，做完搞懂即可。每章题目耗时是不一样的，现在有些忘记当时每章做了几天了，但是按顺序推进就行，大概题目多的章节需要画个4-5天可能，题目少章节的可能一天就能刷完，看个人进度。

**10月-11月初**：我当时先做真题，早期真题一天两张（除非碰到难的年份），个人感觉老题的1998，2001，2003以及新题的2016，2018，2020难度略大，并对答案、把错题搞透（一天3小时差不多）。李林108题，有些地方有些难度，卡住正常，做完搞懂即可（最后只写了高数，线代概统没写）。

**11月中-12月中**：这个时候就是各路模拟卷的狂轰滥炸了，不过23年奇数年会简单一些，建议该年考研不要写李艳芳老师的卷子了，难度比真题大太多了。这个阶段大概一天一张，我刷了李林6、李永乐（6+3）、李艳芳（22的3套+21的3套）、合工大超越5套卷，主要是查缺补漏，分数不重要。个人感觉李林6套卷比较接近考研成绩，这6张卷子除了第一套，都分数在140附近徘徊。

**12月末**：因为政治和英语作文模板的缘故，数学刷题放缓了节奏，李林4套卷分散开找了3天做（第四张没时间做了），其他时间以回顾错题、基础知识的偏门概念（**比如方向导数、梯度、旋度、散度、曲率等**。数一根据21年的特点可能从这里面出题，属于概念略偏但是难度不大的题目）。一天学习数学的时间，除了三次模考一天学习3小时，其他时间一天1小时左右。

**Summary**：对于2022的考研数学卷子，我觉得一方面，大家应该多重视选择题中的概念理解题（论660题的重要性）；另一方面，2022的卷子运算量也比21大一些，建议**加大运算量和运算速度的训练**

## 1.专业课（136）

***资料***： 
王道4本单科书 + 王道真题+王道模拟题+王道新增考点冲刺班（当时花了399，课件比补充文档看着舒服且有条理）+ 政哥4套卷（刘财政老师，当时在中科院群潜水，他给了包含新增考点的4张卷，b站讲解） 

对于遇到的王道书上描述比较模糊的点，我当时参考如下教材：
（1）数据结构：《数据结构与算法分析：C语言描述》第二版，机械工业出版社（**其实根本没看**）；

（2）计算机组成原理：《计算机组成与设计》硬件软件接口，第五版，机械工业出版社；

（3）操作系统：《计算机操作系统》汤小四版，西安电子科技大学出版社；

（4）计算机网络：《计算机网络》谢希仁，第七版，电子工业出版社。

#### 4月-8月 第一轮

4本单科书，看视频课+过一遍王道教材，在其他本子上刷选择题，一门课学完试着把真题的大题做一下，每天学个3-4小时左右就可以；在其他本子上做是为了后面二刷使用

**数据结构**：大三上修读过，所以复习时，有不少内容不用看视频。我当时只有平衡二叉树、哈夫曼树、线索二叉树、KMP算法、B-树、图的应用、各种排序的比较、外部排序的内容看了视频。这个阶段大概耗时20天左右。

**计算机组成原理**：对于王道的课，B站19版视频的讲课我觉得很一般，当时学完后，对于CPU的那一章、IO方式的那一节，还是处于云里雾里的状态，不少东西还是理解不透。幸运的是，后面有人给我推了咸鱼讲的计算机组成原理（我当时以为，他只讲ds和os），在辅助以做题，我感觉思路清晰了不少。王道书兑换码对应的就是上一年的考点精讲，这个计组是**咸鱼老师**讲的，我认为比B站19版的那位老师透彻很多。真题大题卡壳不少，这让我又对存储器、指令系统、CPU这一块的知识重新梳理一遍，脑子里脉络清晰了一些

**计算机网络**：大三下选课选了计网，但是由于课上讲的太高深，理论知识也基本是面向王道复习（期末考试题多数来自王道，只是翻译成英文了）。王道楼楼老师讲得很不错，大题考试的点也分布比较集中，主要是以网络层为载体的综合题居多。计网的知识点，整体感觉比较琐碎，相互之间关联也不是很大，协议一多也容易混乱，建议反复多次看知识点和习题，有时间建议做笔记整理整理（我没整理，主要是时间不够，建议别学我），看多了常见的也就记住了。

**操作系统**：7月中旬开始学习的这门课，感觉知识点本身难度不算大，但是练习题经常错一堆，挺折磨人的。主要是概念的理解出了问题，通过对题目错因的解决，慢慢纠正对概念理解中的误区；此外，题目中经常会出现知识点讲解中没有的东西，这时可以通过课本对应地方来补充知识点（王道书上罗列的知识点并不是特别全面，需要做题补充）。文件系统感觉应该是最难的了，物理结构和逻辑结构里面有很多名词差不多，搞得我头脑比较混乱，直到考前很多词也分不清，但是今年考试题目的文件系统还是难度不大的。

#### 9月-10月底 第二轮

**4本单科书，选择题在书上写，全刷，大题真题部分全部二刷，非真题针对常考点和薄弱点进行多刷（没有全部写完），有的章节不考大题该节的大题我是不做的，比如操作系统的IO系统、计算机网络的物理层等**

二轮刷题后，虽然都是做过的题，感觉有了新的理解与体会，而且对知识点掌握更加牢固了。有些第一轮复习时模模糊糊的东西，比如操作系统的进程管理、内存管理，组成原理的CPU、IO方式等，感觉也清晰了许多。期间历经了专业课大纲大改，当时心态有点崩，甚至有想改考浙软的冲动。**但是想了想，这波吓退一批人，敢继续冲408没准可以捡漏呢，而且自己对于旧大纲内容也掌握的不错了，就继续按照自己既定的计划走下去了**。

#### 11月 第三轮

**真题+4本单科书的薄弱环节+新增考点**

一天刷一张真题（心情好了刷两张）或看一定量的新增考点并做新增考点对应的习题（大概是数据结构1天，网络1天，组成原理4天，操作系统5天）。个人感觉，新增考点习题经常碰上没讲解过的东西，这时候只能自己积累下来了。不过新大纲第一年考的，新增考点就3个选择，难度不大。对于真题里面概念不清引起的错误，建议翻阅王道单科书或前面提到的教材，把那一块的知识点（不是单个知识点）再巩固巩固。

#### 12月 

**模拟题+回扣基础（尤其是新增考点、操作系统和计算机网络）**

王道模拟题难度比近年来真题要略大一些，主要是为了模拟考场环境（卡时间2小时45分钟），限时完成。22版的模拟题里面有一些描述不够严谨的地方，当时我还想着打破砂锅问到底，但经历过考研后，我建议大家放弃这种模糊不清的概念，真题的命题还是比较明确的。

政哥4套卷是包含新增考点的模拟题，难度比王道模拟卷略低，值得一做。

对于这12张卷子不会的题目，建议翻阅王道单科书，把那一块的知识点（不是单个知识点）再巩固巩固。

最后回扣基础大概花了一周，考研第一天前的晚上没有看，考研第二天前的晚上把没有看过的知识看完。

Summary：408难点在于知识点多、琐碎。对我来说比较棘手的是计组和操作系统，因为LZ没有修读过操作系统，计算机组成原理也是从0基础开始看，没做过硬件实验对数据通路的流向等问题，还是掌握起来有难度的。

另一方面，408复习经常出现看了后面的忘了前面的，因此408应该**反复多次**的复习。做题目的时候，一些非真题可能本身有错误，有时候不用纠结那么几个题目，知识点学到即可。在复习的过程中，有些考频为0、且难以理解的知识，建议直接跳过不看，若考前时间充足也可以啃一下，万一出题了呢。
**这些知识包括了：数据结构的外部排序（不包含最小归并树的那一小节），计算机组成原理的定点数除法、硬布线控制器（但是后面那几条特点还是要记一记），计网的移动IP 和IPV6。这些东西，14年来没考过。**

最后，今年的考题有很多**往年真题的影子**，尤其是计网大题，就是以前真题的选择题改个数字、换个问法（比如802.11帧的三个地址）出现在了试卷中。所以，23考研的同学建议真题至少刷两遍（我只刷了一遍吃了点小亏嘤嘤嘤），今年的命题我明显感觉到命题老师**创新点不足了**，考了一些往年真题的改编题。我预测，这个趋势或许会继续下去，因为专业课创新太多容易超纲（盲猜）。所以，出题也越来越回归原来真题的影子了吧。
